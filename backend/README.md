# 🏠 SmartRent Backend - Week 1 Foundation

This is the core API for the SmartRent platform, built with Node.js, Express, and MongoDB. 

## 🚀 Getting Started

### 1. Prerequisites
- Node.js installed
- MongoDB (Local or Atlas)
- Cloudinary Account (for image uploads)
